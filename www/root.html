About This Website

Hey I'm Sean, this is my Hackpage. What's a Hackpage you ask?
A Hackpage is the word I use to define a work in progress website.
A website meant for developers, or will be developers, to start learning how to build a website for themselves.
You would go through the pains of not using tools like sass, or web frameworks to do it for you.
You would instead build on top of this hacky code and realize the pain the tools we use try to ease.
You would then try to slowly convert your code to better states using tools like sass or handlebar templates, or whatever.
